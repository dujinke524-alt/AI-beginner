# AI 初学者学习项目合集

这是我在从零开始学习 AI 过程中的练习项目合集，主要围绕 **结构化输出、文档处理、RAG（检索增强生成）** 进行实践。

本仓库适合和我一样的初学者参考，重点不是做复杂系统，而是通过一个个小项目，真正理解 AI 应用开发中的基础流程。

---

## 仓库内容

本仓库目前包含以下 3 个小项目：

### 1. 结构化报告生成器
将杂乱文本整理为结构化输出，帮助理解提示词设计与 JSON 输出。

### 2. 文档读取与文本切分练习
学习如何读取 Word/PDF 文档，并切分成适合检索的文本块。

### 3. RAG 原文检索 Demo
将文档切块后进行 embedding，存入向量库，并根据用户问题返回最相关的原文内容。

---

## 学习目标

通过这些项目，我希望逐步掌握：

- 文档读取与文本提取
- 文本切分与 chunk 理解
- Embedding 的基本作用
- 向量数据库的基础使用
- 相似度检索
- RAG 的完整基础流程
